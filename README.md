# dotfiles

Personal dotfiles for M2 MacBook Air running Fedora Asahi Remix.

## Stack

- **OS**: Fedora Asahi Remix (aarch64)
- **WM**: Hyprland
- **Package manager**: Nix + home-manager
- **Shell**: zsh

## Usage

```bash
git clone https://github.com/chaewonkong/dotfiles.git ~/.config/home-manager
home-manager switch --flake ~/.config/home-manager
```
